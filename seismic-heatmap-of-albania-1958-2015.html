Seismic Heatmap of Albania, 1958-2015

Wed 20 May 2015

Since there was no detailed Albanian earthquake heatmap on the web, I made this interactive heatmap, depicting all earthquakes (>1 mag) that have hit Albania from 1958 until 2015.

The map was built using Leaflet.js, Leaflet.heat and the data was extracted from USGS Survey.

The geojson data I use can be found here. I use only the latitude, longitude and magnitude (for the intesity parameter), but the USGS website provides many more attributes for each seismic activity.
